Home

Página de inicio, en donde podrá actualizar la base de datos de autorizaciones realizadas.

Documentación

Funciones

Está opción está disponible para inicializar el servidor, restaurando todas las autorizaciones que hayan sido guardadas anteriormente (en caso se haya reiniciado el servidor), o bien, preparando la información necesaria para que puedan procesarse solicitudes, y generar autorizaciones (en caso sea primera vez que interactúa con la página).

Documentación

Cargar Archivo

En esta pantalla, es posible gestionar la carga de los archivos de entrada con extensión .xml con una o varias solicitudes de autorización, para poder ser procesadas.

Documentación

Funciones

Aquí, se podrán cargar los archivos de entrada en formato XML, con las solicitudes o DTE's que se busquen procesar y autorizar. Al momento de seleccionar "Cargar solicitudes", si el archivo subido es un XML, el contenido del mismo se trasladará al área de texto, para que, si lo desea, pueda ser revisado y editado, antes de mandarlo a procesar.

Una vez se decida procesar las solicitudes ingresadas en el área de texto, se mandaran todas ellas al servidor. Al terminado el proceso, se mostrará en pantalla una ventana emergente, indicando las solicitudes recibidas, así como las facturas correctas o incorrectas encontradas (las aprobaciones se realizará únicamente a las facturas correctas).

Consultar Datos

Al consultar esta página, se muestra la información almacenada en la base de datos de todas las autorizaciones realizadas. El contenido de esta vista es el mismo que se puede encontrar en el archivo XML de autorizaciones.

Documentación

Funciones

Se muestran todas las autorizaciones seaparadas por fechas, en donde, al seleccionar la fecha de interés, se muestra un resumen de todas las facturas recibidas, los errores detectados, así como un listado de las aprobaciones realizadas en esa fecha.

Al seleccionar esta opción, automáticamente se restaura el servidor a su estado inicial. Sin datos. Debe tomarse en cuenta que esta acción no puede deshacerse.

Documentación

Resumen de IVA por fecha y NIT

Al seleccionar esta opción, podrá elegir la fecha por la cual se desea filtrar la gráfica, mostrando un resumen de movimientos por NIT en la fecha escogida. Por cada NIT que se encuentre registrado en esa fecha, se podrá apreciar el IVA emitido y el IVA recibido.

Documentación

Funciones

En este apartado, se podrá escoger la fecha en la que se desea ver el resumen de movimientos por NIT, además de la opción de graficar el resumen, y generar un PDF con la gráfica (para generar el PDF, se debe generar la gráfica primero, para verificar que existen movimientos en la fecha seleccionada).

Finalmente, por cada NIT registrado, se observará el IVA emitido en color verde, y el recibido en color azul.

Documentación

Resumen por rango de fechas

Al seleccionar esta opción, se podrá elegir un rango de fechas por la cual se requiere filtrar los montos totales, luego, podrá elegir si desea ver los valores totales con o sin IVA incluido. Además, se podrá visualizar por cada fecha que se encuentre dentro del rango elegido, el valor autorizado total (con o sin IVA).

Documentación

Funciones

En este apartado, se podrá escoger el rango de la fecha en la que se desea ver el resumen de montos totales por fecha, y escoger visualizar los montos totales con o sin IVA. Además, tendrá la opción de graficar el resumen, y generar un PDF con la gráfica (para generar el PDF, se debe generar la gráfica primero, para verificar que existen movimientos en la fecha seleccionada).

Finalmente, por cada fecha que haya registrado movimientos, se observará el monto total en color verde.
